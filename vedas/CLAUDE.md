# Vedas Read-Book Project

Multilingual Vedic scripture readers following `../alice-in-wonderland/` patterns.
All HTML files are self-contained (no external JS/CSS beyond Google Fonts).

## Readers

### 1. `reader.html` — Multi-Veda curated reader (scrolling)
- 4 Vedas (Rigveda, Samaveda, Yajurveda, Atharvaveda)
- 13 suktas, 95 mantras — curated famous hymns
- Bengali-script Sanskrit (primary), Devanagari (toggleable), IAST (toggleable)
- Bengali + English meanings
- 5 themes (sepia, light-purple, light-azure, dark-violet, dark-blue)
- Alice-pattern top bar, sidebar, settings (font/size/width)

### 2. `fullbleed.html` — Multi-Veda book-spread reader
- Same 4-Veda content as reader.html
- Two-page spread: Sanskrit left, meanings right
- Page-flip navigation, touch swipe, theme picker

### 3. `rigveda.html` — Complete Rigveda compact reader
- **All 1,028 suktas, 10,143 mantras** from 10 mandalas
- Generated by `build_rigveda.py` (fetches from rigveda-online.github.io, CC0)
- Two-column layout: translations (L) | Sanskrit (R)
- Bengali-script Sanskrit + Devanagari (same font size, toggleable)
- English (Griffith 1896) + Bengali meanings (49 hand-curated, rest TBD)
- Mandala tabs + horizontal sukta strip + full grid overlay
- Infinite scroll within mandala
- Alice-pattern controls: 4 themes, font, size, width, devanagari toggle
- Hash routing: `#3.62` for bookmarks

## Data sources

### Curated (`seeds/`)
- `hymns.json` — Rigveda (6 suktas, 49 mantras) with Bengali meanings
- `samaveda.json` — 2 suktas (Agni invocation, Kena Upanishad Ch.1)
- `yajurveda.json` — 3 suktas (Purnamadah, Isha Upanishad, Shiva Sankalpa)
- `atharvaveda.json` — 2 suktas (Prithivi Sukta, Kala Sukta)

### Generated
- `seeds/rigveda-complete.json` — Full Rigveda (10,143 mantras), human-readable JSON
- `rigveda-samhita.md` — Full Rigveda as Markdown

### Build pipeline (`build_rigveda.py`)
1. Fetches 1,028 sukta HTML files from `rigveda-online.github.io` (CC0)
2. Parses Devanagari Sanskrit + Griffith English per verse
3. Transliterates Devanagari → Bengali script (Unicode offset +0x80)
4. Overlays hand-curated Bengali meanings from `seeds/hymns.json`
5. Outputs: `rigveda.html`, `seeds/rigveda-complete.json`, `rigveda-samhita.md`
6. Caches fetched HTML in `/tmp/rigveda_cache/` (re-run < 30s)

## Design notes

- Bengali-script Sanskrit is the primary reading experience
- Devanagari at same font size, toggleable via settings or D key
- alice-in-wonderland/reader.html is the reference for all UI patterns
- `|` = pada separator (inline), `||` = verse end (line break)
- Titles from Griffith's convention (deity names)
- Default theme: light-purple
